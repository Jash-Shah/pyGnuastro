******************************
Overview of the Python Modules
******************************

The Gnuastro Python Package is split into separate modules. Each of these modules can be imported using ``import gnuastro.MODULE_NAME`` in your Python code. Each module aims to provide similar functionality as its `Gnuastro Library`_ counterpart.
