pyGnuastro Documentation¶

pyGnuastro is a free software which offer a wide array of functions for astronomical data manipulation and analysis. It is the Python implementation of GNU Astronomy Utilities and aims to offer all the functionality of Gnuastro but with a higher level of abstracion.
